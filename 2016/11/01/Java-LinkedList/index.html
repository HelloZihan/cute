Java集合之LinkedList

发表于 2016-11-01 更新于 2016-12-20 分类于 Java
LinkedList底层是链表，随机访问速度相对较慢，因为其要一步步移动指针来定位；但是执行插入或删除时，`LinkedList`较快，因为其直接指针修改即可。

前面写过ArrayList，此类继承了RandomAccess这个标记接口，标记其随机访问速度快(例如使用get方法，set方法等通过索引直接定位的方法)，因为其底层是数组，所以直接通过索引定位速度较快。而LinkedList底层是链表，随机访问速度相对较慢，因为其要一步步移动指针来定位；但是执行插入或删除时，LinkedList较快，因为其直接指针修改即可，而ArrayList要重新移动数据来填补空间。因此在遍历时，LinkedList使用迭代器速度会较快，而通过索引值去访问会比较慢，而ArrayList与此相反。
